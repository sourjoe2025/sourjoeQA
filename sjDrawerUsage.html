sjDrawer Usage

This page demonstrates four different drawer directions using the same core open_drawer(...) API. Each button opens a different drawer with a different content theme.

How this module works

  1. Keep source title/content blocks in the page, usually hidden.
  2. Call open_drawer(whichDrawer, drawerType, titleNode, contentNode, optionalHeight).
  3. The module moves those real nodes into the active drawer.
  4. When the drawer closes, the nodes are restored to their original hidden container.
Click the grey overlay or the × button in the drawer header to close. This page intentionally shows the four main directions separately so future coders can copy the pattern they need.
